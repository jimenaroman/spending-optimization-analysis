# spending-optimization-analysis
# Spending Optimization & Financial Decision Insights

This project analyzes an estimated six-month personal spending dataset to identify major cost drivers, distinguish fixed and variable expenses, evaluate financial stability, and model realistic cost-reduction strategies.

The goal is not just to track general expenses, but to generate decision-oriented insights and recommend practical interventions.

## Tools Used
- Python
- pandas
- NumPy
- Matplotlib
- Jupyter Notebook
